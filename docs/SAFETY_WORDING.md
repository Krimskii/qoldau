# Safety Wording — Qoldau AI

> Полный список запрещённых и разрешённых формулировок в продукте.
> Назначение: гарантировать, что Qoldau AI не позиционируется как медицинский инструмент.

## Главный принцип

> **Qoldau AI — это инструмент наблюдений, а не медицинский инструмент.**
> Продукт не диагностирует, не лечит, не заменяет специалиста.

## Разрешённые формулировки (по контексту)

### AI-наблюдение

✅ «Похоже, …»
✅ «Возможно, …»
✅ «Похоже, … связано с …»
✅ «Может быть связано с …»
✅ «Часто подтверждалось как …»

### Предупреждения

✅ «Это наблюдение, не диагноз.»
✅ «Нужно подтвердить наблюдениями.»
✅ «Можно обсудить со специалистом.»
✅ «Точность зависит от количества наблюдений.»

### Действия

✅ «Можно попробовать …»
✅ «Можно отметить в профиле»
✅ «Можно попробовать короткую паузу»
✅ «Можно предложить тактильный инструмент»

### Confidence

✅ «Высокая уверенность»
✅ «Средняя уверенность»
✅ «Низкая уверенность»
✅ «Подтверждено N раз»
✅ «Подтверждено N взрослыми»

### Статусы

✅ «Подтверждено»
✅ «Нужно проверить»
✅ «Исправлено»
✅ «Отклонено»
✅ «Черновик»

## Code review checklist

Перед мержем проверять:

- [ ] Нет слов: лечит, диагностирует, исправляет, нормализует, нарушение, патология, манипулирует.
- [ ] Нет утверждений «точно», «определённо», «безусловно».
- [ ] Все AI-выводы начинаются с «Похоже…» или «Возможно…».
- [ ] Все suggestions начинаются с «Можно попробовать…».
- [ ] На каждом экране есть disclaimer.
- [ ] Нет обращений «врач», «доктор» — только «специалист».
- [ ] Confidence не подаётся как истина — только как «N подтверждений».
