# PRODUCT_BRIEF.md

## Qoldau AI — Product Brief

### Vision

Voice-first AI-платформа, которая помогает семьям и специалистам сопровождать детей с РАС. Ребёнок даёт сигнал → взрослый говорит → AI структурирует → система помогает видеть закономерности.

### Core Principle

**Не медицинское устройство.** Не диагностирует, не лечит, не заменяет специалиста. Все AI-выводы — гипотезы.

### User Roles

1. **Ребёнок** — AAC карточки, голосовой ввод, выбор любимых
2. **Родитель** — голосовые наблюдения, подтверждение событий, аналитика
3. **Тьютор** — наблюдения, отчёты родителям
4. **Специалист** — ABC-анализ, паттерны, формирование отчётов

### Main Flow

1. Ребёнок даёт сигнал (звук, жест, карточка)
2. Сопровождающий говорит наблюдение голосом
3. STT превращает речь в текст
4. AI структурирует текст в события
5. Взрослый подтверждает
6. Система строит единую ленту событий, профиль коммуникации и отчёты

### Key UX Principles

1. **Voice-first для взрослых** — не заполнять формы, а говорить
2. **Visual-first для ребёнка** — крупные карточки, минимум текста
3. **Human-in-the-loop** — AI предлагает, человек подтверждает
4. **Hypothesis language** — "Похоже...", "Возможно...", "Нужно подтвердить"

### MVP Scope

- Voice-first ввод для родителя и тьютора
- STT API (mock на первом этапе)
- Event Timeline — единая лента событий
- AAC карточки для ребёнка
- Базовые события: питание, туалет, сон, поведение, коммуникация
- AI summary (mock)
- Human confirmation flow

### Phase 2

- Распознавание звуков ребёнка
- Wearable устройство
- Геозоны
- Домашний хаб
- Персональные AI-модели

### Design Language

- Светлый, мягкий, современный
- Много воздуха, карточный UI
- Teal/turquoise как primary
- Rounded corners (24px)
- Крупные кнопки для ребёнка
- Без "больничного" вида

### Compliance Notes

- Не медицинское устройство
- Не делает диагнозов
- Не заменяет специалиста
- Все выводы — наблюдения/гипотезы
