
## Задачи
1. `src/data/atec.ts` — структура анкеты ATEC (4 подшкалы: речь/коммуникация,
   социализация, сенсорика/когнитивность, здоровье/поведение) — пункты RU.
   (Использовать существующий бесплатный RU-перевод пунктов.)
2. `src/pages/parent/AtecCheckIn.tsx` — прохождение анкеты (раз в 4–8 недель),
   результат → событие `type:'measure'` в ленту (payload: подшкалы + суммарный балл, дата).
3. `ParentAnalytics` — график динамики ATEC по времени (line chart из событий `measure`).
   Меньше = лучше; **без причинных заявлений** («изменения во времени», не «улучшение из-за X»).
4. Напоминание: если прошло >8 недель с последнего — мягкая подсказка пройти снова.
5. Дисклеймер под результатом: «Это шкала самонаблюдения, не диагноз. Обсудите со специалистом.»
